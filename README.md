# Supplementary Material for "Hybrid Task Planning and Scheduling in Heterogeneous Multi-Agent Systems based on LLMs and Constraint Programming"


This repository contains the supplementary material of the "Hybrid Task Planning and Scheduling in Heterogeneous Multi-Agent Systems based on LLMs and Constraint Programming" paper, in which we present a two-layer architecture that combines the benefits of Constraint Programming (CP) with the flexibility of Large Language Models (LLMs).
